Free Soundtracks you can activate in Steam

This is a list I've compiled myself activating each soundtrack so I know by the current creation time (27-Oct-2022) they all work. As far as I know, they are all free and legal to add to your Steam library even if you don't have the base game. Actually some don't even have base game. Others are simply not listed in the base game page (e.g. the Postal ones, what a surprise for me!)

So not only you can have a lot of new, legal music to try, specially if you're into game soundtracks; you might discover that you could have had a soundtrack for a game you already have, but you didn't see because devs made a mistake, or who knows.

Remember that you can remove a product from your library (not just uninstall it) if you don't like it.

Caveats:
* YOU CAN ONLY ACTIVATE 50 PRODUCTS EACH HOUR. More products will fail to be added, and each try will reset the cooldown to a whole hour. You've been warned.
* Some free soundtracks are missing, because I forgot to put in another list the ones I already have. I can't test myself if they are free to be added without a really bothering method. They shouldn't be many, because I got many free soundtracks due to limited promotions, having the base game, getting them before they needing the base game, etc. Example of one I think it's free: https://store.steampowered.com/app/1224060/World_of_Warships__Composers_Choice/
* Some might need the base game to be added, but I already had the base game so I don't know if they can be activated without the base game, as I explained above. Very few cases.
* I advice you to check the games from the soundtrack's page itself. You'll find nice surprises you might want to add to your wishlist, and also several good/excellent rated free games.
* Sometimes adding a soundtrack to your library will add the free base game too (Flat Trip, Curvatron, maybe more).
* Not all are soundtracks. There are audiobooks and drama CDs too.
* If by the time you open a store page, it redirects you to the Steam Store homepage, or says "this is not available anymore in Steam", you'll most probably not be able to add it, tho I think with maybe one or two I was lucky.

Instructions:
For most cases: simply open the pages (in Firefox, there's a nice and useful extension called "Multipaste and Go button" which allows you to open several URLs at once. In each page, simply push the blue "Add to library" button. If everything goes well, after a second or two, you'll see a box saying "Whatever Game Soundtrack has been added to your account. It is now available in your Steam Library." If you push the OK button, the page will refresh saying that it's in your library.
If that's not the message you got, read it, you might have activated already 50 products in the last hour and that's why it's failing. Other times it needs the base game, saying "Before you can add this product to your Steam account, you must own the base product."
If it says you got it, but you don't actually see anything added to your client library, exit the Steam client and restart it. It'll probably fix the problem and everything new in your library should appear.
Sometimes it'll open the client and redirect you to the Steam Store homepage, or again to the soundtrack page. Then the soundtrack can't be added by the normal method.
A few times it'll open a simple error page saying that you need the base game (instead of the message box mentioned above) or another error. These soundtracks change over time, sometimes devs change the requirements to get them: price, owning base game, regions, who knows.
In very few cases, that blue button to "Add to library" will be missing, e.g. with https://store.steampowered.com/app/1284270/Defective_Holiday_OST/
No problem, use the first steps in the following section and activate it through SteamDB.

Extreme case instructions, if you're patient:
Sometimes the normal method will not work. In rare cases, using SteamDB can work, tho it's not guaranteed at all; actually it only worked for few few cases for me. Note that for SteamDB to be able to see what you own and not, you need to log in if you haven't done (it's through OpenID, it's safe and doesn't need any extension). Otherwise, just to trying to install, it's not needed to be logged in SteamDB.
First of all, transform the Steam address into a SteamDB address. You only need to put the AppID number in the right place of this URL: https://steamdb.info/app/AppIDGoesHere/
Example:
https://store.steampowered.com/app/1224060/World_of_Warships__Composers_Choice/
https://steamdb.info/app/1224060/
Go to the SteamDB page of that AppID. You'll see some buttons at the top right, including one called "Install". Press it and see if it's added to your library in your client. The button will also change from "Install" to "Owned". If not, it'll take a bit of time for the client to give a timeout error, close it.
If installing from the AppID page didn't work, you can still try the last option, tho I don't remember it ever worked. I might be wrong so I'll explain it. At the left menu, press on the "Packages" list. You'll see all the SubIDs where the soundtrack is included. Open each one of those pages (on the "SUBID" column) and click "Install" in each of those pages.
If everything fails, wait one whole hour without trying to activate any product in Steam, and retry the normal instructions. If it still fails, then for some reason the soundtrack can't be activated anymore.
