# ejercicio integrador

"""
Una universidad desea crear un programa para contabilizar los cursos que
tiene cada alumno. Para ello se debe realizar primero una aplicación de
consola la cual debe solicitar el nombre de un alumno y la cantidad de
cursos en la que se encuentra inscripto.

Estos dos valores deben almacenarse como una lista de dos elementos (el 
nombre y la cantidad de cursos como un número entero) en una lista de 
alumnos.

Una vez hecho esto, se debe hacer que el programa, al iniciar, pregunte
cuál de las siguientes dos operaciones se debe realizar: ingresar un
alumno o ver la lista de alumnos ingresados. Un ejemplo de lo que debe 
aparecer en consola, en una posible implementación, es lo siguiente:

Ingrese el número de la operación que desea ejecutar:
1 - Ver la lista de alumnos.
2 - Añadir un alumno a la lista.
3 - Salir.
>>> 2
Ingrese el nombre del alumno: Pablo
Ingrese la cantidad de cursos: 3
¡El alumno fue añadido a la lista!

Ingrese el número de la operación que desea ejecutar:
1 - Ver la lista de alumnos.
2 - Añadir un alumno a la lista.
3 - Salir.
>>> 1
Lista de alumnos:
Pablo - 3 cursos

Ingrese el número de la operación que desea ejecutar:
1 - Ver la lista de alumnos.
2 - Añadir un alumno a la lista.
3 - Salir.
>>> 4
La opción ingresada no es correcta, vuelva a
intentarlo.
Ingrese el número de la operación que desea ejecutar:
1 - Ver la lista de alumnos.
2 - Añadir un alumno a la lista.
3 - Salir.
>>> 3
¡Gracias por utilizar el programa!

"""
# Estrategia de resolución:
# 1) construir el template while True
# 2) Diagramar mi estructura de datos

# alumnos = [ ["Juan",3], ["Ana",4], ["Victoria",2] ]

alumnos = []

while True:
    
    print("""
            Menu de opciones
    --------------------------------
    1 - Ver la lista de alumnos.
    2 - Añadir un alumno a la lista.
    3 - Salir.
    --------------------------------
    """)

    opcion = input("Seleccione una opción: ")
    
    if opcion == "1":
        if not alumnos:
            print("No hay alumnos")
        
        else:
            print("Lista de alumnos:")
            for alumno in alumnos:
                print(f"{alumno[0]} - {alumno[1]} cursos")
            
                
    elif opcion == "2":
        nombre = input("Nombre: ")
        while True:
            cursos = input("Cursos: ")
            if cursos.isdecimal() and int(cursos) != 0:
                break
            else:
                print("Error: debe ingresar un entero positivo")
                   
        alumnos.append([nombre, cursos])
        print("¡El alumno fue añadido a la lista!")
        
        
    elif opcion == "3":
        print("Gracias por usar este programa ...")
        break
        
    else:
        print("Opción incorrecta")
