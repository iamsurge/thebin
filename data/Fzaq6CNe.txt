Turing: teoría de funciones computables ¿cuales son las funciones

TURING-completo: es un lenguaje de programación que puede escribir

Un lenguaje Turing completo necesita al menos tres estructuras 
lógicas:
1 condicional, 1 bucle indefinido, 1 bucle definido

# En Python existen solo 3

# condicional: if

# Script que pide una edad y responde su condición frente al voto

edad = int(input("Ingrese su edad: "))

if 0 < edad < 16:
    print("Voto prohibido")
    
elif 16 <= edad < 18  or edad >= 70:
    print("Voto optativo")

elif 18 <= edad < 70:
    print("Voto obligatorio")
    
else:
    print("Error en el ingreso de la edad")


# bucle indefinido: while se usa cuando no se cuantas veces debo repetir 
# algo

# Script que pide un nro y lo muestra en pantalla. Se debe finalizar 
# cuando se ingrese un cero

while True:
    numero = int(input("Ingrese un nro: "))
    if not numero:
        break
    print(numero)


# bucle definido: for
alumnos = ["Hugo", "Ana", "Orlando"]

for alumno in alumnos:  # for <iterador> in <iterable>
    print(alumno)

# Colección: TODA COLECCION ES ITERABLE:. Colecciones: listas, tuplas, 
# diccionarios, cadenas

# Script que cuenta las vocales de una frase

    if caracter in "aeiouAEIOU":
        total_vocales += 1 # total_vocales = total_vocales + 1
        
print(f"Total vocales: {total_vocales}")
