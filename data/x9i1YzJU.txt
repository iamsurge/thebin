When data is bound by the same name immutably, it freezes.
```rust
fn main() {
    let mut _mutable_integer = 7i32;
    {
        // Shadowing by immutable name; froze.
        let _mutable_integer = _mutable_integer;
        _mutable_integer += 1; // panic
    }
    _mutable_integer += 1; // no longer frozen.
}
