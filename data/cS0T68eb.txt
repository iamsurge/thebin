User & Group:
change a users password: passwd <username>
create user and assign group: useradd -G <group> <username>
list all groups: getent group
list groups for a user: id or id <username/groupname> or groups
create a group: groupadd <groupname>
add  user to a group: usermod -a -G <groupname> <username>
